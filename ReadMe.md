## 一、运行

要运行，必须先安装依赖，使用命令:

```cmd
npm i --registry=https://registry.npm.taobao.org
```

即可安装依赖。

然后通过命令:

```cmd
npm start
```

即可跑起来看效果了。 当然 [点击这里](http://test.microanswer.cn/wzwgame/index.html) 可以直接查看线上效果。
